# Available Data
 • ISO 3166 compliant country names<br>
 • Official country denominations (full name)<br>
 • ISO 3166-1 alpha 2 code<br>
 • ISO 3166-1 alpha 3 code<br>
 • ccTLD (top level domain) code<br>
 • international phone code (predial)<br>

# Available Formats
 • JSON<br>
 • XML<br>
 • CSV<br>
